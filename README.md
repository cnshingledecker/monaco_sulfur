# monaco_sulfur

This repository contains MONACO source plus sulfur-focused model setups.

Included model directories:

- `ralf_network_2025/`: sulfur network/model setup associated with the 2025 Ralf network work.
- `one_phase_model_b_validation/`: self-contained one-phase dense-cloud validation case that reproduces the archived paper-era `Model_B` `bS8` curve exactly.
